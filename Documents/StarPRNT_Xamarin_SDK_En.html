1.4. Connecting a Printer to a Device

1.4.1. Bluetooth

See the Online manual (Setup - Connect Tablet/PC) for how to connect mC-Print2, mC-Print3 or TSP100IIIBI to a device.
All printers have each initial device name as a factory default setting, such as "Star Micronics". When using multiple devices, which have same device name, it is useful to change the device name for identifying each device easily.

  • You can change the bluetooth settings of the printer, such as changing the bluetooth device name. Please use the Printer Configuration Software.
1.4.1.1. Pairing of a bluetooth printer with your device (iOS)

  1. Turn the printer power.

  2. When the security setting of the printer is set to SSP, press the PAIR button for more than 5 seconds to make it available for pairing.

  3. Tap [Settings] > [Bluetooth].

  4. Tap bluetooth to turn it on. Your iOS device searches and displays the Bluetooth devices in range. Tap the printer you want to pair with.

  5. If a PIN code is used for printer pairing, enter the PIN and tap Pair. The factory default is “1234”. (If use the “SPP”, do not need this operation.)

  6. When the pairing is complete, you’ll see this message.

  • You can change the iOS Port Name of the printer. Please use the Printer Configuration Software.

  • To confirm iOS Port Name, select [Settings]-[General]-[About] after bluetooth pairing is established. The iOS Port Name will be shown under the bluetooth address.
1.4.1.2. Pairing of a bluetooth printer with your device (Android)

  1. Turn the printer power.

  2. When the security setting of the printer is set to SSP, press the PAIR button for more than 5 seconds to make it available for pairing.

  3. Tap Settings.

  4. Tap [Bluetooth] to ON.

  5. Tap [Search for Devices].
    The Bluetooth devices are scanned and the found devices are displayed. Find the printer you wish to connect to and tap it to pair.

  6. If a PIN code is used for printer pairing, enter the PIN and tap Pair. The factory default is “1234”. (If use the “SPP”, do not need this operation.)

1.4.1.3. Pairing of a bluetooth printer with your device (Windows)

  1. Turn the printer power.

  2. When the security setting of the printer is set to SSP, press the PAIR button for more than 5 seconds to make it available for pairing.

  3. Select [Settings] > [Devices].

  4. Select [Bluetooth & other devices].

  5. Tap [Bluetooth] to ON.

  6. Select [Printers & scanners] > [Add a printer or scanner].

  7. Select the Bluetooth device name, and select [Add a device].

  8. If a PIN code is used for printer pairing, enter the PIN and tap Pair. The factory default is “1234”. (If use the “SPP”, do not need this operation.)

  9. When the device appears in “Printers & Scanners”,pairing is completed.

1.4.2. Ethernet

See the Online manual (Setup - Connect Tablet/PC) for how to connect mC-Print2, mC-Print3 or TSP100IIILAN to a device.
All printers ship with DHCP enabled by default. If your network supports DHCP, be sure to make the necessary configurations so that your printer will automatically get an IP Address.
Use your printer with the #9100 Multi Session disabled. The setting can be confirmed by Test Print which can be executed by holding down the printer’s feed button while turning the printer on.
This software uses TCP communication port 9100, TCP communication port 9100, UDP communication port 2222.

  • Refer to "guidelines-ethernet_en.pdf” for how to confirm and change the #9100 Multi Session setting and how to set the Static IP Address.

  • You can set up your POS Printers which can connect to TCP/IP. Please use the Printer Configuration Software.
    TSP100IIIUtility and Star Setting Utility cannot be used with printers where an IP address has not been assigned (IP address 0.0.0.0).
1.4.2.1. Connecting a ethernet printer to your device (iOS)

  1. Assign an IP Address to the printer and connect it to the network. .Tap Settings.

  2. Ensure Wi-Fi is ON.

  3. Connect to the same network the printer is on.

1.4.2.2. Connecting a ethernet printer to your device (Android)

  1. Assign an IP Address to the printer and connect it to the network.

  2. Tap Settings.

  3. Ensure Wi-Fi is ON.

  4. Connect to the same network the printer is on.

1.4.2.3. Connecting a ethernet printer to your device (Windows)

  1. Assign an IP Address to the printer and connect it to the network.

  2. Connect to the same network the printer is on.

1.4.4. USB

No specific Star Micronics driver or printer software needs to be installed on the Android device.

1.4.4.1. Connecting a USB printer to your device (iOS)

  1. Connect the printer and the iOS device with a Lightning cable.

  • How to debug the USB printer
    When using a USB printer, because the iOS device Lightning port is used for the printer connection, it is not possible to install apps or debug using the Lightning cable. However, if the following conditions are satisfied, wireless debugging can be used.

    • macOS 10.12.4 or later

    • The latest version of Visual Studio for Mac

    • Xcode 9.0 or later

    • iOS 11.0 or later

*The following descriptions, procedures are subject to changes when versions upgrade. Refer to the Microsoft web site for the most recent information.

  1. The iOS device and mac must be connected to the same network.

  2. Set a pass code for the iOS device using the iOS device “Settings” app.

  3. Connect the iOS device and mac using the Lightning cable.

  4. Launch Xcode and select "Window” - "Devices and Simulators".

  5. Select the iOS device to use in the left pane, and place a checkmark next to "connect via network".

  6. Thereafter, Visual Studio will display the iOS device in the network even when it is not connected using the Lightning cable, allowing app installation and debugging.
1.4.4.2. Connecting a USB printer to your device (Android)

  1. Connect the printer and the Android device with a USB cable.

  • The USB cable needed can vary by device. Review the specifications for your device.

  • When connecting, multiple devices using a USB hub, the recommended cascade connection is maximum one stage. The maximum number of devices which can be connected at the same time is 4.

3.1. Printing flow using a IPort interface

Using the IPort interface, print per the following procedure.

Sample Code
IPort port = null;
IPrinterStatus printerStatus;

try
{
    // Port open
    port = Port.GetPort(portName, portSettings, timeout);

    // Print end monitoring -Start
    printerStatus = port.BeginCheckedBlock();

    // Status judgment before printing
    if (printerStatus.Offline)
    {
        // Printing offline (no paper, printer cover open etc)
        return;
    }

    DateTime startDateTime = DateTime.Now;

    int total = 0;

    while (true)
    {
        // Send print data
        int written = port.WritePort(commands, total, commands.Length - total);

        total += written;

        // Completed sending of all print data
        if (total >= commands.Length)
        {
            break;
        }

        TimeSpan timeSpan = DateTime.Now - startDateTime;

        if (timeSpan.TotalMilliseconds >= 30000)
        {
            // Timeout error
            return;
        }
    }

    // Print end monitoring - Timeout setting
    port.EndCheckedBlockTimeoutMillis = 30000;

    // Print end monitoring -End
    printerStatus = port.EndCheckedBlock();

    // Status judgment at end of print end monitoring
    if (printerStatus.Offline)
    {
        // Printing offline (no paper, printer cover open etc)
        // Printing is abnormal termination
    }
    else
    {
        // Printer online
        // Print successful end
    }
}
catch (PortException exception)
{
    // Error
}
finally
{
    // Port close
    if (port != null)
    {
        Port.ReleasePort(port);
        port = null;
    }
}
